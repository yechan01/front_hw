C

1972년 미국의 벨 연구소에서 데니스 리치(Dennis Ritchie)가 개발한 범용 프로그래밍 언어

(운영 체제 개발을 목적으로 만들어졌으며, 특히 유닉스(UNIX) 운영 체제의 개발에 핵심 역할을 했다.)

주요 특징

  1. 절차 지향 언어: C는 순차적으로 명령을 실행하는 절차 지향적 특성을 가지고 있습니다. 프로그램이 함수로 나누어져 실행됩니다.
  2. 저수준 접근 가능: C는 하드웨어와 가까운 저수준 언어이면서도 고수준 언어의 기능을 제공하여, 메모리 관리나 포인터를 통해 하드웨어에 직접 접근할 수 있습니다.
  3. 이식성: C로 작성된 프로그램은 다양한 플랫폼에서 실행할 수 있어 이식성이 뛰어납니다.
  4. 풍부한 표준 라이브러리: 다양한 입출력, 수학 계산, 문자열 처리 등을 위한 표준 라이브러리를 제공하여 개발 편의성을 높입니다.
  5. 포인터와 메모리 관리: C 언어는 포인터 개념을 통해 메모리를 직접 다룰 수 있으며, 효율적인 프로그램 작성이 가능합니다. 하지만 메모리 관리를 개발자가 직접 해야 하므로 주의가 필요합니다.
  6. 컴파일 언어: C는 컴파일러를 통해 기계어로 번역되어 실행됩니다. 따라서 C 프로그램은 빠르고 효율적으로 실행됩니다.

활용 분야

  • 운영 체제 개발 (유닉스, 리눅스 등)
  • 임베디드 시스템 (소형 기기, 마이크로컨트롤러)
  • 시스템 소프트웨어 (컴파일러, 인터프리터)
  • 게임 개발 및 그래픽 프로그래밍
